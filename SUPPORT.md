# Support for Deucarian Logging

Use [GitHub Issues](https://github.com/Deucarian/Logging/issues) for reproducible logging-facade, category, sink, or editor-tooling defects; documentation gaps; and narrowly scoped feature requests. Search existing issues before opening a new one.

Include the package version, Unity editor version, configured sinks/categories, the smallest reproduction, expected and actual output, and relevant stack traces with sensitive log content removed.

`main` is the stable package channel. `develop` is a preview channel and may change before promotion. Please reproduce against the applicable current channel before reporting an older snapshot.

Do not paste secrets, personal data, proprietary logs, or vulnerability details into an issue. Follow [SECURITY.md](SECURITY.md) for private security reports.

Support is provided on a best-effort basis. Telemetry, upload pipelines, and diagnostics products outside the logging boundary are out of scope.
